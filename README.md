# Akhilesh Shinde - Interactive Developer Portfolio 🚀

Welcome to the source code of my personal developer portfolio! This project was built to go beyond a standard resume, acting as an interactive, AAA-game-inspired experience that showcases my skills in modern web development, 3D graphics, and UI/UX design.

🌐 **Live Site:**
```bash
https://akhilesh-portfolio-three.vercel.app/
```

---

## ✨ Extraordinary Features

This portfolio is packed with advanced interactive elements:

- **Interactive 3D Character:** A fully integrated Three.js / WebGL character in the hero section that reacts to user interactions with custom GSAP animations.
- **Custom Developer Terminal:** A fully functional, hidden command-line interface accessible via the `>_` icon. Try running commands like `whoami`, `projects`, `skills`, or `clear`!
- **Cinematic Transitions:** AAA-game style loading screens, noise overlays, and smooth reveal animations powered by GSAP.
- **Premium Carousel Layout:** A custom CSS-Grid horizontal scroll section showcasing my top projects inside sleek MacOS browser mockups.
- **Silky Smooth Scrolling:** Integrated `@studio-freight/lenis` for buttery smooth scrolling mechanics across all devices.

---

## 🛠️ Tech Stack

- **Framework:** React 18 with TypeScript + Vite
- **Styling:** Custom Vanilla CSS (for maximum granular control and performance)
- **3D & Rendering:** Three.js, React Three Fiber, React Three Drei
- **Animation:** GSAP (GreenSock Animation Platform)
- **Physics:** Matter.js *(Zero-gravity interactive tech stack - coming soon)*

## 🚀 Running Locally

To run this project on your local machine:

1. Clone the repository:
   ```bash
   git clone https://github.com/akhishinde2004/akhilesh-portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd akhilesh-portfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

> **Note on GSAP:** This project uses GSAP for complex animations. Ensure you have the appropriate GSAP licenses if you intend to fork and host this commercially.
